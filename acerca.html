Objetivo General

Identificar como es un sitio web con navegacion interactiva, por medio de AJAX, por consiguiente no se necesita cargar la pagina para pasar de una pagina a otra, mas que solo trae la informacion que se solicita al momento de hacer click en cierto enlace.

Tecnologias

Esta web fue realizada con las tecnologías de PHP, Javascript, HTML, CSS y AJAX, sin embargo, este ultimo no es una tecnología si no un conjunto de técnicas que permiten que las aplicaciones web funcionen de forma asíncrona.
